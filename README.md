# How to open GradeHub Website
## install framwork
- pip install flask in terminal
- pip install Jinja2
 or install in EXTENSIONS find "Jinja"
## run Website
- run in app.py
- Then press CTRL + Click,there will be 2 links, click one of them.

# วิธีเปิดเว็บไซต์ GradeHub
## ดาวน์โหลดเฟรมเวิร์ค
- เปิด terminal และพิมพ์ pip install flask
- เปิด terminal และพิมพ์ pip install Jinja2หรือดาวน์โหลดใน EXTENSIONS และค้นหา "Jinja"
## เปิดเว็บไซต์
- กด run ที่ไฟลล์ app.py
- จากนั้นกด CTRL + Click,จะมี 2 ลิงค์ กดที่ลิงค์อันใดอันหนึ่ง
